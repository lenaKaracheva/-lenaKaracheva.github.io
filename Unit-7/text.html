    Установим с помощью Font размер, шрифт и цвет текста

      Тег basefont для указания цвета текста, размера шрифта для всего текста в документе HTML
      Тег i одержимое внутри обычно отображается курсивом
      Тег ins определяет текст, который был вставлен в документ. Браузеры обычно подчеркивают вставленный текст.
      Тег small определяет меньший текст (например, копирайт и другие побочные комментарии)
      Тег b выделяет жирный текст без какой-либо дополнительной важности.
      Тег strong используется для определения текста высокой важности. Содержимое внутри обычно отображается полужирным шрифтом
      Тег sup определяет верхний индекс текста
      Тег mark для выделения части текста желтым цветом
      Тег sub определяет текст нижнего индекса
      Тег del определяет текст, который был удален из документа. Браузеры обычно перечеркивают удаленный текст
      Тег code используется для определения фрагмента компьютерного кода
      Тег pre 
      определяет предварительно отформатированный текст.
